# Cómo Recibir Donaciones

## Opción 1: PayPal (Fácil, pero requiere cuenta)

### Pasos:
1. Ve a https://www.paypal.com
2. Crea cuenta (gratis)
3. Verifica tu cuenta con email
4. Activa PayPal.Me:
   - Ve a https://www.paypal.me/settings
   - Elige tu nombre de usuario: paypal.me/TUNOMBRE
5. Pon este link en README.md y LICENSE

### Ventajas:
- Fácil de usar para quien dona
- Acepta tarjetas de crédito
- Usado mundialmente

### Desventajas:
- Requiere tu nombre real para verificación
- Comisión del 2-3% por transacción
- Necesitas cuenta bancaria vinculada

---

## Opción 2: Bitcoin (Máxima privacidad, sin datos personales)

### Pasos:
1. Descarga Electrum: https://electrum.org
2. Instala en tu PC
3. Crea una "Standard Wallet"
4. Escribe las 12 palabras de recuperación (GUÁRDALAS EN LUGAR SEGURO)
5. Ve a "Receive" tab
6. Copia tu dirección Bitcoin (empieza con "bc1..." o "1..." o "3...")
7. Pon esta dirección en README.md y LICENSE

### Ventajas:
- Totalmente anónimo (no requiere datos personales)
- Sin comisiones para ti
- No necesitas cuenta bancaria
- Nadie puede bloquearte

### Desventajas:
- Menos gente usa Bitcoin
- Precio volátil (puedes perder/ganar valor)
- Tienes que convertir a dinero fiat (€/$) por ti mismo

---

## Opción 3: Ko-fi (Recomendado si quieres privacidad)

### Pasos:
1. Ve a https://ko-fi.com
2. Crea cuenta con email
3. Puedes usar un alias (no tu nombre real)
4. Vincula PayPal o Stripe (ellos gestionan pagos)
5. Tu link será: ko-fi.com/TUNOMBRE
6. Pon este link en README.md y LICENSE

### Ventajas:
- No requieres PayPal para quien dona
- Interfaz bonita
- Sin comisión para ti (solo 3% de procesamiento)
- Puedes usar alias

### Desventajas:
- Menos conocido que PayPal
- Igual necesitas PayPal o cuenta bancaria para retirar dinero

---

## Opción 4: GitHub Sponsors (Profesional, integrado)

### Pasos:
1. Ve a tu perfil GitHub → Settings → Sponsors
2. Activa GitHub Sponsors
3. Vincula Stripe o PayPal
4. Aparecerá botón "Sponsor" automático en tu repositorio

### Ventajas:
- Integrado directamente en GitHub
- Muy profesional
- GitHub no cobra comisión

### Desventajas:
- Requiere cuenta bancaria verificada
- No puedes usar alias (necesitas datos reales)

---

## Mi Recomendación

**Para máxima privacidad**: Bitcoin (Electrum)
**Para facilidad**: PayPal
**Mejor balance**: Ko-fi + Bitcoin

### Configuración Recomendada:

1. Crea wallet Bitcoin en Electrum (5 minutos)
2. Crea cuenta PayPal (10 minutos)
3. Actualiza README.md y LICENSE con:

```markdown
## Apoyar el Proyecto

Si SitioSeguro te ha sido útil:

**PayPal**: paypal.me/TUNOMBRE
**Bitcoin**: bc1q...tu_direccion...

Gracias por tu apoyo.
```

---

## Importante

- NUNCA compartas tus 12 palabras de Bitcoin (quien las tenga, roba tus fondos)
- Guarda las palabras en papel, no en PC
- Para PayPal, usa contraseña fuerte + 2FA
- Para Bitcoin, haz backup de tu wallet

---

## Preguntas Frecuentes

**¿Cuánto cuesta recibir donaciones?**
- Bitcoin: 0% (pero pagas comisión de red al mover fondos, ~$1-5)
- PayPal: 2-3% por transacción
- Ko-fi: 3% de procesamiento

**¿Necesito declarar donaciones?**
Depende de tu país y cantidad. En general:
- España: < 10,000€/año no declarar
- Consulta con un contador si recibes mucho

**¿Puedo poner cualquier dirección Bitcoin?**
Sí, pero DEBES tener acceso a ella. Si pones una aleatoria, pierdes el dinero.

**¿PayPal mostrará mi nombre real?**
Sí, cuando alguien done verá tu nombre de cuenta PayPal.
